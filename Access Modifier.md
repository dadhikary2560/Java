#### Public Class



Accessibility- Accessible to all, even if in different project

Can be applied with- class, method, variable





#### Private Class



Accessibility- within the same class

Can be applied with- Inner class, method, variable





#### Default Class



Accessibility- within same package

Can be applied with- class, method, variable





#### Protected Class



Accessibility- same package or different package if it's child class

Can be applied with- Inner class, method, variable
